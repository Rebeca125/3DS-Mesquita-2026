Dedicamos o projeto BookForge a alunos e professores que fazem da educação um caminho de transformação. Desenvolvido para facilitar o acesso ao conhecimento, o sistema torna o uso da biblioteca mais prático e organizado.
Trabalho apresentado à Escola como requisita
à conclusão da matéria Projeto Multidisciplinar em Desenvolvimento de Sistemas
sob orientação da Prof. Sara Silva dos Santos

1.0 Apresentação da Empresa:
O projeto “BookForge” é um Projeto de Desenvolvimento Web com foco em Sistema de Gerenciamento de Biblioteca (Library Management System). O site permite que os usuários possam consultar o catálogo de livros, façam reservas online e receba confirmações para retirada física na biblioteca.
Inclui funcionalidades como busca por título/autor/gênero, verificação de disponibilidade em tempo real, gerenciamento de contas de usuário (login via e-mail), notificações por e-mail/SMS sobre reservas e devoluções.
